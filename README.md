# Real-Time Emotion Detector

This project is a real-time facial emotion detection system built using **Python** and **OpenCV**.  
It identifies basic human emotions like **Happy, Sad, Angry, Fear, Surprised,** and **Neutral** using Haar Cascade classifiers.
